# Contributing

Thanks for your interest in contributing.

## Development setup
1. Create and activate a virtual environment.
2. Install dependencies from requirements.txt or pyproject.toml.

## Code style
- Keep changes focused and well-scoped.
- Add brief comments only when logic is non-obvious.
- Prefer clear function and variable names.

## Testing
- Add or update tests when changing logic.
- Include a short note on how you validated changes.

## Pull requests
- Describe what changed and why.
- Link to relevant issues if any.
- Include a minimal repro or steps to verify.
